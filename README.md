# Chatbot using LangGraph 🤖

A conversational AI chatbot built using [LangGraph](https://python.langchain.com/docs/langgraph) to manage stateful, multi-actor workflows. This project demonstrates how to create an intelligent, context-aware chatbot capable of handling complex conversational flows using graph-based logic.

## 🌟 Features
* **Stateful Conversations:** Retains context across multiple turns using LangGraph's state management.
* **Agentic Workflows:** (Optional: Mention if your bot uses tools, e.g., web search, calculators, or APIs).
* **Flexible LLM Integration:** Powered by LangChain, allowing easy swapping of underlying models (e.g., OpenAI, Anthropic, open-source models).

## 🛠️ Tech Stack
* **Python 3.9+**
* **[LangGraph](https://python.langchain.com/docs/langgraph)**
* **[LangChain](https://python.langchain.com/)**
